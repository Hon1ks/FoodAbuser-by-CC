## Бесплатные модели

### ★ Gemini Flash — лучший бесплатный вариант

| Параметр | Значение |
|----------|----------|
| Лимит | **1 500 запросов/день** |
| Точность распознавания | 90–92% |
| Русская кухня | Хорошо |
| Возвращает КБЖУ | Да (через промпт) |
| Платный тариф | ~$0.0002/запрос (в 25× дешевле GPT-4o) |
| Формат | base64 или URL |

**Подключение:**
```ts
provider: 'gemini',
apiKey: 'AIza...'  // Google AI Studio — бесплатно
```

---

### ★★ Groq + Llama 3.2 — лучший по лимиту

| Модель | Лимит/день | Точность | КБЖУ |
|--------|-----------|----------|------|
| Llama 3.2 90B | 3 500 | 88–90% | Частично |
| Llama 3.2 11B | **7 000** | 80–85% | Слабо |

- OpenAI-совместимый API — подключается как GPT-4o, только другой URL
- Рекомендуется: Groq распознаёт блюдо → CalorieNinjas выдаёт КБЖУ

**Подключение:**
```ts
provider: 'groq',
apiKey: 'gsk_...'  // api.groq.com — бесплатно
```

---

### Ollama + LLaVA — self-hosted (безлимитно)

| Модель | Точность | Требования |
|--------|----------|-----------|
| LLaVA 7B | ~78% | 8GB VRAM / 16GB RAM |
| LLaVA 13B | ~84% | 16GB VRAM / 32GB RAM |
| LLaVA 34B | ~88% | 40GB VRAM |

- Нет лимитов, нет стоимости API, полная приватность
- Требует GPU сервер для нормальной скорости
- Качество ниже облачных решений

---

### HuggingFace Food Models — не рекомендуется

- Только 101 класс (западная кухня, Food-101 датасет)
- **Русские блюда отсутствуют** — борщ, пельмени, плов не распознаёт
- Нестабильный бесплатный лимит (~300/час, с "засыпанием")
- Не возвращает КБЖУ

## Рекомендуемая стратегия по стадиям

### Стадия 1 — MVP ($0/мес)
```
Основной:  Gemini Flash       → 1500 запросов/день бесплатно
Fallback:  Groq 90B           → 3500 запросов/день бесплатно
КБЖУ:      CalorieNinjas      → 10 000 запросов/мес бесплатно
```
Покрывает ~1000 пользователей с 2–3 приёмами пищи в день.

### Стадия 2 — Ранний продакшен (~$3–15/мес)
```
Основной:  Gemini Flash платный  → $0.0002/запрос
Резервный: GPT-4o               → $0.005/запрос (сложные блюда)
КБЖУ:      Nutritionix           → бесплатно до 500 req/день
```

### Стадия 3 — Масштаб ($50–500/мес)
```
Основной:  GPT-4o Vision
Дешёвый:   Gemini Flash (простые/повторные запросы)
КБЖУ:      Nutritionix Pro / Edamam
```

## Как подключить в проекте

Файл конфигурации: `constants/aiConfig.ts`

```ts
// MVP — бесплатно:
provider: 'gemini',
apiKey: 'AIza...'

// Или платный лучший:
provider: 'openai',
apiKey: 'sk-...'

// Или рекомендованный комбо:
provider: 'openai_nutritionix',
apiKey: 'sk-...',
appId: 'nutritionix_app_id',
nutritionixKey: 'nutritionix_key',
```

Полный список провайдеров и их адаптеры — в `services/aiService.ts`.
